# Password.spec.md

## ✅ 有効なケース

- "Abcd1234"
- "StrongPass1"
- "A1b2c3d4"

## ❌ 無効なケース

| Input       | Reason                          |
|-------------|---------------------------------|
| "abc"       | 短すぎる                       |
| "abcdefgh"  | 大文字と数字が含まれていない         |
| "ABCDEFGH"  | 小文字と数字が含まれていない         |
| "abcd1234"  | 大文字が含まれていない                    |
| "ABCD1234"  | 小文字が含まれていない                    |
| "Abcdefgh"  | 数字が含まれていない                       |
