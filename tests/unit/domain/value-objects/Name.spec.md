# Name.spec.md

## ✅ 有効なケース

- "田中太郎"
- "John Doe"
- "鈴木"
- "Li"
- "José María"

## ❌ 無効なケース

| Input       | Reason                          |
|-------------|---------------------------------|
| ""          | 空文字列は無効                  |
| "a"         | 1文字は短すぎる                |
| "   "       | 空白のみは無効                  |
| " a "       | トリム後に1文字になるため無効  |
| "!@#$"      | 記号のみは無効                  |
